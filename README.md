# Dashboard App (Python Shiny)

Дашборд аналитики данных на базе **Python Shiny**. Миграция с R Shiny на Python.

## Требования

- Python 3.10+

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск приложения

```bash
shiny run app.py --port 8000
```

После запуска приложение будет доступно по адресу: **http://127.0.0.1:8000**

## Использование

1. Откройте **http://127.0.0.1:8000** в браузере
2. Загрузите Excel-файл с данными (лист "Пациенты")
3. Используйте боковую панель для фильтрации:
   - **Группа пациентов** — выбор группы
   - **Город** — мульти-выбор городов
   - **Дата взятия образца** — диапазон дат
   - **Возраст** — диапазон возраста
4. Просматривайте данные на вкладках:
   - **Дашборд** — карта, таблицы распределения, диаграммы
   - **Набор данных** — полная таблица с данными
